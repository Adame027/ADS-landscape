O que é o GitHub?

O GitHub é uma plataforma online que permite armazenar, versionar e compartilhar código-fonte usando o sistema de controle de versão Git.

É uma ferramenta essencial para qualquer desenvolvedor, pois permite:

  • Guardar o histórico completo de alterações no código
  • Trabalhar em equipe em um mesmo projeto
  • Exibir o portfólio de projetos para recrutadores
  • Contribuir com projetos open source
